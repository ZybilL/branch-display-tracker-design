Design Handoff

Branch Product Display Tracker

An internal tool for a 14-branch retail chain. Branch staff photograph their current in-store product display and log which item needs to be replaced with which new item. A designer reviews and approves/rejects each submission, and approved/rejected jobs are compiled into a report for the schematic (visual merchandising) team.

This gallery shows the six screens as rendered reference images.

These are static reference screenshots, not interactive pages. The original design files (.dc.html) were built in a prototyping tool and require that tool's runtime to render — they aren't included here. See the handoff README.md in the source repo for the full spec, data model, and design tokens.
Source design files, data model, and full spec are documented in README.md alongside this bundle.
